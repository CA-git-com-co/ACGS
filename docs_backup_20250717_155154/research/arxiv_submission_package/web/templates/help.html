Getting Started

Learn how to use the Academic Submission Tool to validate and optimize your research papers.

Quick Start Guide

  1. Prepare your files: Gather your LaTeX source files, bibliography, figures, and any supporting code.
  2. Upload your submission: Use the upload page to select and upload your files.
  3. Choose validation type: Select full validation for comprehensive checking or quick check for basic issues.
  4. Review results: Examine the validation report and follow the recommendations.
  5. Fix issues: Address any problems identified in the validation process.
  6. Re-validate: Upload your improved submission for final verification.

Validation Categories

Checks for common LaTeX syntax errors including:

  • Unmatched braces and brackets
  • Undefined references and citations
  • Missing packages and commands
  • Proper document structure

Ensures bibliography completeness and quality:

  • Required fields for different entry types
  • Proper BibTeX formatting
  • Citation consistency
  • DOI and URL validation

Verifies figure references and quality:

  • All referenced figures exist
  • Proper figure formats (PNG, PDF, EPS)
  • Figure captions and labels
  • Resolution and size requirements

Checks compliance with venue-specific requirements:

  • arXiv: File size limits, format requirements
  • IEEE: Template compliance, reference formatting
  • ACM: Style guidelines, accessibility standards

Troubleshooting

Common Issues
  • Upload fails: Check file size (max 50MB) and supported formats
  • LaTeX errors: Ensure all packages are included and syntax is correct
  • Missing figures: Verify figure files are in the correct directory
  • Bibliography issues: Check BibTeX syntax and required fields
